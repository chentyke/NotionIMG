大屏幕优化测试页面

优化说明

这个页面展示了针对不同屏幕尺寸的优化效果：

  • 1200px以上： 增加容器宽度到1400px，字体大小适度增加
  • 1600px以上： 容器宽度扩展到1600px，字体进一步增大
  • 2048px以上： 容器宽度达到1800px，针对2K及以上屏幕优化

页面列表优化

页面列表页面针对大屏幕优化了网格布局、字体大小和卡片间距，提供更好的浏览体验。

阅读页面优化

文章阅读页面增加了文本字体大小、行间距和容器宽度，提高了长文本的可读性。

代码示例

以下是大屏幕优化的CSS代码示例：

@media (min-width: 1200px) {
    .container {
        max-width: 1400px;
    }
    
    .prose {
        font-size: 1.25rem;
        line-height: 1.8;
    }
    
    .page-title {
        font-size: 3rem;
    }
}
