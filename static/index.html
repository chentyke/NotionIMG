Found in your Notion database URL

Instructions

1. Create a Notion Integration

Visit Notion Integrations and create a new integration. Copy the integration token.

2. Share Your Database

Open your Notion database, click "..." menu, select "Add connections" and choose your integration.

3. Get Database ID

Copy the database ID from your database URL (the part between the workspace name and "?")

4. Configure Database

Make sure your database has:

  • Name (Title) property
  • type (Text) property set to "image"
  • Content (Files) property with your image
